# Robin helper
## Rosan
Rosan is the integrated tool for spectrum analisis
It performs graphical signal for Time and Frequency domains

## Installation
You will need certain python modules
make sure you have pip for python3 and run:
> ./install_modules.sh


## Usage

Open rosan:

> ./rosan
